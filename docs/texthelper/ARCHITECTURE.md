# Текстовый помощник — архитектура

Описание инструмента «Текстовый помощник»: назначение, структура кода, интеграция.

---

## Назначение

Набор утилит для работы с текстом. Открывается по кнопке «Текстовый помощник» в навигации.

### Инструменты (7)

| Инструмент | Описание |
|------------|----------|
| **Столбец → Строка** | Объединяет строки в одну через выбранный разделитель (запятая, пробел, точка с запятой и др.) |
| **Строка → Столбец** | Разбивает строку на отдельные строки по разделителю |
| **Удалить дубликаты** | Оставляет уникальные строки, показывает отчёт |
| **Удалить пустые строки** | Убирает пустые строки из текста |
| **Добавить обёртку** | Добавляет текст в начало/конец каждой строки (кавычки, скобки, нумерация) |
| **Убрать пробелы** | Удаляет лишние пробелы (по краям, двойные) |
| **Изменить регистр** | ВЕРХНИЙ, нижний, Каждое Слово, Первое слово, иНВЕРТИРОВАННЫЙ |

## Структура

```
tools/texthelper/
├── app.js              # Точка входа: initTextHelper(), открытие/закрытие модалки
├── logic/
│   ├── columnToRow.js      # Столбец → Строка
│   ├── rowToColumn.js      # Строка → Столбец
│   ├── removeDuplicates.js # Удаление дубликатов
│   ├── removeEmpty.js      # Удаление пустых строк
│   ├── prefixSuffix.js     # Добавление обёртки
│   ├── trim.js             # Удаление пробелов
│   ├── changeCase.js       # Смена регистра
│   └── stats.js            # Статистика (символы, слова, строки)
├── ui/
│   └── texthelperUI.js     # UI: рендеринг инструментов, настроек, обработчики
└── css/
    └── texthelper.css      # Стили модалки и компонентов
```

---

## Интеграция

- **index.html:** кнопка `#texthelper-btn` в навигации, модалка `#texthelper-modal-overlay`
- **main.js:** импорт `initTextHelper` из `tools/texthelper/app.js`
- CSS: `tools/texthelper/css/texthelper.css` подключен в index.html
